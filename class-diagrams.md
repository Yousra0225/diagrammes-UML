# Diagramme de classe
```bash
+----------------------+
| NomClasse |
+----------------------+
| - attribut1 : Type |
| - attribut2 : Type |
+----------------------+
| + méthode1() : Type |
| + méthode2() : Type |
+----------------------+
```

- `+` → public
- `-` → private
- `#` → protected
- `: Type` → type de retour ou type d’attribut
